사용설명서

최종 업데이트: 2026-02-15

1. 첫 설정(온보딩)

  • 처음 실행 시 이름/소개를 입력한 뒤 역할을 선택합니다.
  • 타자: 수비 포지션을 선택하면 프로필/AI 분석 기준이 해당 포지션으로 저장됩니다.
  • 투수: 투수 유형, 주력 구종(최대 5개), 릴리스 포인트, 투구 손을 모두 선택해야 다음 단계로 이동됩니다.
  • 설정 완료 후 프로필 홈에서 언제든 `포지션 수정`으로 다시 변경할 수 있습니다.
온보딩 중 앱 설치가 자동으로 안 뜨면 정상입니다. iPhone은 `홈 화면에 추가` 방식으로 설치해야 합니다.

2. 경기 기록 입력(핵심)

  • 입력 탭에서 시즌, 날짜, 경기명, 결과, 컨디션을 먼저 입력합니다.
  • 기록 유형은 `타자 기록 / 투수 기록` 버튼으로 전환합니다.
  • 필수에 가까운 값은 핵심 기록 영역에서 입력하고, 필요 시 `세부 기록 펼치기` 버튼으로 추가 항목을 입력합니다.
  • 경기 저장 버튼은 신규 저장과 수정 저장을 같은 위치에서 처리합니다.

3. 타자 입력 규칙

  • 타석(PA)을 입력하면 타수(AB)는 자동 계산됩니다.
  • 계산식: AB = PA - BB(볼넷) - HBP(사구), 최소 0으로 보정됩니다.
  • 타율은 자동 계산된 AB와 안타(H) 기준으로 반영됩니다.
  • 2루타, 3루타, 도루 등은 세부 기록에서 입력할 수 있습니다.
PA를 먼저 입력하면 시즌 통계/프로필 `총 타석` 수치가 정확하게 누적됩니다.

4. 투수 입력 규칙

  • 이닝은 `5`, `5.1`, `6.2` 형식으로 입력합니다.
  • 내부 계산은 아웃카운트 기반으로 처리되어 ERA/누적 이닝 계산 오차를 줄입니다.
  • 실점(RA), 자책점(ER), 탈삼진(SO), 피안타, 볼넷 허용, 피홈런을 입력하면 핵심 지표가 자동 계산됩니다.
  • 평균/최고 구속은 구종별로 입력할 수 있으며 AI 분석에도 반영됩니다.
예: 5.1이닝 = 5이닝 1아웃, 5.2이닝 = 5이닝 2아웃

5. 통계 탭 읽는 법

  • 상단 `타자 템플릿 / 투수 템플릿` 전환으로 역할별 지표를 분리해서 볼 수 있습니다.
  • 경기별 그래프는 단기 변동, 누적 그래프는 시즌 흐름 확인에 적합합니다.
  • 주간 리포트는 최근 경기 기준으로 핵심 수치를 요약해줍니다.

6. 캘린더/멘탈 탭 활용

  • 캘린더 탭: 날짜별 경기 결과와 상세 요약을 빠르게 복기할 수 있습니다.
  • Mental 탭: `잘된 선택/흔들린 이유/다음 목표` 구조로 남기면 다음 경기 루틴 설계에 유리합니다.
  • 멘탈 기록은 단기 메모가 아니라 시즌 루틴 데이터로 누적됩니다.

7. 프로필 탭 활용

  • 프로필 홈: 시즌 지표, 뱃지, 최근 흐름, 갤러리 확인
  • 프로필 수정: 이름/소개/프로필 이미지/갤러리 변경
  • 포지션 수정: 역할/포지션(또는 투수 설정) 재지정
  • 기록 관리: 최근 기록 수정/삭제, 시즌 리셋, 신규 사용자 초기화

8. AI SCOUT REPORT 해석 가이드

  • AI 페이지는 현재 시즌 + 현재 프로필 역할 기준으로 분석합니다.
  • 타자/투수 역할에 따라 점수 기준과 분석 문구가 달라집니다.
  • 최근 경기 데이터가 적으면 분석 신뢰도가 낮아질 수 있습니다.
  • 포지션/투수 유형 변경 후에는 새 기준으로 다시 확인하세요.

9. 자주 묻는 문제 해결

  • 설치 안내가 안 보일 때: 홈화면 앱 캐시를 지우거나 Safari/Chrome에서 새로고침 후 다시 시도
  • 광고/외부 리소스 경고: 외부 도메인 robots 정책 경고는 앱 자체 차단과 다를 수 있음
  • 입력값이 이상할 때: `기록 관리`에서 해당 경기 수정 후 다시 저장
  • UI가 예전 버전으로 보일 때: PWA 캐시 갱신(앱 재실행/재설치) 진행
