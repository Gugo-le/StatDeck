# StatDeck

브라우저에서 동작하는 야구 기록/분석 웹앱입니다.
타자/투수 기록을 입력하고, 시즌 통계와 캘린더, 프로필, AI 분석(예측/스카우트 점수)을 확인할 수 있습니다.

## 주요 기능

- 타자/투수 기록 입력
  - 타자: 타수, 안타, 2/3루타, 홈런, 볼넷, 사구, 타점, 득점, 도루
  - 투수: 이닝 아웃카운트, 실점/자책, 피안타, 볼넷/사구, 피홈런, 탈삼진
  - 투수 구속: 구종별 평균속도 + 최고속도 입력
- 온보딩
  - 신규 사용자 닉네임/소개
  - 타자: 수비 포지션 선택
  - 투수: 운영 역할, 구종, 릴리스 포인트, 좌/우완 선택
- 통계/시각화
  - 시즌 지표(타자 템플릿 / 투수 템플릿)
  - 경기별/누적 차트
  - 주간 리포트
  - 캘린더 + 날짜별 상세 기록(수정/삭제)
- Mental Entry
  - 짧은 멘탈 메모 저장/조회
- 프로필
  - 프로필 편집, 아바타/사진 업로드, 기록 관리
- AI 분석 페이지
  - 타자/투수 역할별 모델 분리
  - 다음 경기 예측 + SCOUT SCORE + 유형 분석
  
## 데이터 저장

- 기본 저장소: `IndexedDB` (`statdeck-db`)
- 주요 키
  - `seasonData`
  - `profileData`
  - `quotes`
  - `lastSeasonKey`
- AI 모델
  - `indexeddb://statdeck-model-batter-{season}`
  - `indexeddb://statdeck-model-pitcher-{season}`

## AI 동작 요약

- 최소 5경기 이상일 때 분석 수행
- 타자: 타율/ISO 기반 예측 + 점수화
- 투수: ERA/WHIP/삼진율/구속(평균/최고) 기반 예측 + 점수화
- 데이터 해시가 변경되면 재학습, 동일하면 저장 모델 재사용

## 신규 사용자로 다시 시작하기

- 브라우저 개발자도구에서 `IndexedDB`의 `statdeck-db` 삭제
- 또는 앱 내 기록/프로필 리셋 기능 사용

## 참고

- 분석 결과는 로컬 기록 기반 참고 지표입니다.
- 의료/법률/금융처럼 고위험 의사결정 용도가 아닌, 퍼포먼스 트래킹 보조 용도로 사용하세요.
