# StatDeck

브라우저에서 동작하는 야구 기록/분석 웹앱입니다.
타자/투수 기록을 입력하고, 시즌 통계, 캘린더, 프로필, AI 분석(예측/스카우트 점수)을 확인할 수 있습니다.

## 주요 기능

- 입력(모바일 최적화)
  - 핵심 입력 + 접기/펼치기 구조로 부담을 줄인 경기 입력 UI
  - 타자: 타석, 안타, 2/3루타, 홈런, 볼넷, 사구, 타점, 득점, 도루 (타수 자동 계산)
  - 투수: 이닝 아웃카운트, 실점/자책, 피안타, 볼넷/사구, 피홈런, 탈삼진
  - 투수 구속: 구종별 평균속도 + 최고속도 입력
  - 경기 컨디션 입력(좋음/보통/나쁨)
- 온보딩
  - 신규 사용자 닉네임/소개 입력
  - 타자: 수비 포지션 선택(필드 UI)
  - 투수: 운영 역할, 구종, 릴리스 포인트, 좌/우완 선택
- 통계/시각화
  - 시즌 지표(타자 템플릿 / 투수 템플릿)
  - 경기별/누적 차트
  - 주간 리포트
  - 캘린더 + 날짜별 상세 기록(수정/삭제)
- 테마
  - 모래톤 라이트 모드 / 다크 모드 전환 (설정 저장)
- Mental Entry
  - 멘탈 메모 저장/조회
- 프로필
  - 앱 첫 진입 기본 탭: 프로필
  - 프로필 편집, 아바타/갤러리 업로드
  - 역할별 업적 뱃지 + 연속기록 노출
  - 기록 관리/리셋/신규 사용자 초기화
- AI 분석(`analytics.html`)
  - 타자/투수 역할별 모델 분리
  - 다음 경기 예측 + SCOUT SCORE + 유형 분석


## 데이터 저장

- 기본 저장소: `IndexedDB` (`statdeck-db`)
- 주요 키
  - `seasonData`
  - `profileData`
  - `quotes`
  - `lastSeasonKey`
- AI 모델 저장 경로
  - `indexeddb://statdeck-model-batter-{season}`
  - `indexeddb://statdeck-model-pitcher-{season}`

## AI 동작 요약

- 최소 5경기 이상일 때 분석 수행
- 타자: 타율/ISO 기반 예측 + 점수화
- 투수: ERA/WHIP/삼진율/구속(평균/최고) 기반 예측 + 점수화
- 데이터 해시가 변경되면 재학습, 동일하면 저장 모델 재사용

## 신규 사용자로 다시 시작하기

- 프로필 > 기록 관리 > `신규 사용자로 되돌리기`
- 또는 개발자도구에서 `IndexedDB`의 `statdeck-db` 삭제

## 참고

- 분석 결과는 로컬 기록 기반 참고 지표입니다.
